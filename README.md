# pomodoro
Aplicativo desktop que aplica o método pomodoro para estudo.

O método pomodoro consiste em dividir o tempo para estudo em blocos de 25 minutos, e intercalar esses blocos de estudo com blocos de 5 minutos para descanso. Após intercalar 3 vezes totalizando 1 hora e 30 minutos de estudo, é recomendado fazer uma pausa mais longa, de 30 minutos, e então repetir o ciclo de duas horas. 
O aplicativo que está sendo disponibilizado faz com que o ciclo de duas horas seja marcado sonoramente e de maneira que possa ser resetado ou pausado. 

Espero que possa servir também para demonstrar para novos programadores em C uma forma de tocar arquivos de áudio.
